## Quick Test (5 Minutes)

### 1. Start System
```bash
START_SYSTEM.bat
```
Open: http://localhost:5173 (or 5174)

### 2. Test as STUDENT
1. Login: student1@test.com / password123
2. Click "Book Resource"
3. Select any resource
4. Fill form and submit
5. Click "My Bookings" - see your booking

### 3. Test as STAFF
1. Logout
2. Login: staff1@test.com / password123
3. Click "Booking Approvals"
4. Approve the student's booking
5. Click "All Bookings" - see approved booking
